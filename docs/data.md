# Fonte de dados

## Grocery Inventory and Sales Dataset (Kaggle)
Autor **Salahuddin Ahmed**. 
**Disponível em**: [link](https://www.kaggle.com/datasets/salahuddinahmedshuvo/grocery-inventory-and-sales-dataset/data)

**Descrição:**  

Este é um conjunto de dados fictício foi projetado para fins educacionais ou de demonstração, fornecendo informações tabulares detalhadas sobre vários itens de supermercado.

Inclui detalhes do produto, informações do fornecedor, níveis de estoque, dados de reabastecimento, preços e desempenho de vendas para 990 produtos em diversas categorias.

**Características Principais:**

    • Expiration_Date: A data de validade do produto.
    • Last_Order_Date: A última data em que o produto foi encomendado.
    • Warehouse_Location: O local de armazenamento do produto.
    • Sales_Volume: Volume total de unidades vendidas.
    • Inventory_Turnover_Rate: Taxa na qual o produto vende e é reabastecido.
    • Status: Status atual (por exemplo, Ativo, Descontinuado, Em falta).
